# CheeseVisuals
Всякие визуалы из СНГ читов, а так же другие прикольные функции в удобной форме мода (нет, тут нету самих читов)

## Описание
Cкорее всего вы когда нибудь видели моды типа TopkaVisuals или PulseVisuals. Это - абсолютно то же самое, бесплатно, с почти всеми функциями, которые есть там, плюс куча другой хрени, которую обычно устанавливают с помощью других модификаций.

## Список функций
### Combat
No Friend Damage - Не отправляет пакет удара, если пытаешься ударить друга.\n

### Movement
AutoSprint - Постоянно нажимает на Ctrl\n
FreeLook - Отключает вращение персонажа на F5\n

### Player
AutoAccept - Автоматически пишет /tpyes для друзей.\n
AutoRespawn - Автоматически жмет "возродиться".\n
CustomCooldown - Позволяет настроить визуальный кулдаун на предметы.\n
ItemScroller - Буквально айтемскроллер.\n
MiddleClickFriend - Позволяет добавить друга на СКМ.\n
NoWorkbench - Добавил для своего удобства, отключает взаимодействие с верстаками, ибо я задолбался на них нажимать на своей базе.\n

### Render
Arrows - Рисует стрелочки. (возможно будет удалено в будущем, бесполезно)\n
AspectRatio - Позволяет менять разрешение экрана прямо в игре.\n
BlockHighlight - Меняет подсветку блока.\n
Breadcrumbs - Спавнит круги там, где идет персонаж.\n
Crosshair - Позволяет настроить прицел.\n
ExtraTab - Увеличивает количество отображаемых игроков в табе.\n
Fullbright - Позволяет видеть в темноте (просто гамму на 10000 в settings.txt).\n
ItemPhysics - Буквально физика выпавшим предметам.\n
JumpCircles - Кружочек под ногами при прыжке.\n
LittlePet - Спавнит собаку на стороне клиента (ходит за игроком).\n
Nametags - Более читаемые никнеймы. (возможно будет удалено в будущем, бесполезно)\n
NoRender - Отключает огонь.\n
Particles - Настраиваемые партиклы.\n
Prediction - Показывает куда падает снаряд.\n
ShulkerPreview - Показывает, что в шалкере.\n
SwingAnimations - Меняет анимацию удара/использования.\n
Trails - Трейл за игроком.\n
ViewModel - Позволяет менять расположение левой и правой рук на экране.\n
World - Позволяет настроить время в мире на стороне клиента.\n

### Misc
ClientSounds - Прикольные звуки при нажатии кнопок.\n
DeathCoords - Птправляет координаты смерти в чат.\n
DiscordRPC - Показывает плашку в дискорде, что вы используете [вставьте название]визуалс.\n
FTHelper - Отображает задержки на кастомные предметы фантайма, позволяет забиндить кнопки на хотбар для использования этих предметов.\n
HWHelper - Отображает задержки на кастомные предметы холиворлда, позволяет забиндить кнопки на хотбар для использования этих предметов.\n
RWHelper - Ультра WIP.\n
IRC - Позволяет подключаться к IRC чату (хз зачем оно вообще здесь, айпи и ключ можно задать в IrcManager.class).\n
NameChanger - Меняет никнейм на стороне клиента.\n
NoCommands - Отключает настройку через команды.\n
Optimizer - Занимается чисткой утечек ОЗУ. Должен повышать производительность на слабых ПК.\n
ServerRPSpoof - Отключает загрузку серверного ресурспака.\n
Unlink - Отключает мод полностью.\n

## Полезные команды
c$config - Позволяет сохранить конфиг.\n
с$macro - Позволяет забиндить команду на кнопку.\n
с$bind - Позволяет забиндить функцию на кнопку.\n
c$irc - Позволяет настроить IRC.\n

## Сборка из сурсов
1. Откройте папку с исходными файлами в IntelliJ IDEA
2. Нажмите на "Build", для того, чтобы докачались все необходимые файлы
3. Зайдите в Gradle -> Tasks -> build -> build
4. Ваш собранный дистрибутив CheeseVisuals будет находиться в [директория проекта]/build/libs/[название мода].jar
