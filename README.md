# CheeseVisuals
Всякие визуалы из СНГ читов, а так же другие прикольные функции в удобной форме мода (нет, тут нету самих читов)
Эдакое "все в одном", чтобы можно было поставить один единственный мод вместо целой сборки.
## Описание
Cкорее всего вы когда нибудь видели моды типа TopkaVisuals или PulseVisuals. Это - абсолютно то же самое, бесплатно, с почти всеми функциями, которые есть там, плюс куча другой хрени, которую обычно устанавливают с помощью других модификаций.

## Список функций
### Combat
No Friend Damage - Не отправляет пакет удара, если пытаешься ударить друга.

### Movement
AutoSprint - Постоянно нажимает на Ctrl

FreeLook - Отключает вращение персонажа на F5

### Player
AutoAccept - Автоматически пишет /tpyes для друзей.

AutoRespawn - Автоматически жмет "возродиться".

CustomCooldown - Позволяет настроить визуальный кулдаун на предметы.

ItemScroller - Буквально айтемскроллер.

MiddleClickFriend - Позволяет добавить друга на СКМ.

NoWorkbench - Добавил для своего удобства, отключает взаимодействие с верстаками, ибо я задолбался на них нажимать на своей базе.

### Render
Arrows - Рисует стрелочки. (возможно будет удалено в будущем, бесполезно)

AspectRatio - Позволяет менять разрешение экрана прямо в игре.

BlockHighlight - Меняет подсветку блока.

Breadcrumbs - Спавнит круги там, где идет персонаж.

Crosshair - Позволяет настроить прицел.

ExtraTab - Увеличивает количество отображаемых игроков в табе.

Fullbright - Позволяет видеть в темноте (просто гамму на 10000 в settings.txt).

ItemPhysics - Буквально физика выпавшим предметам.

JumpCircles - Кружочек под ногами при прыжке.

LittlePet - Спавнит собаку на стороне клиента (ходит за игроком).

Nametags - Более читаемые никнеймы. (возможно будет удалено в будущем, бесполезно)

NoRender - Отключает огонь.

Particles - Настраиваемые партиклы.

Prediction - Показывает куда падает снаряд.

ShulkerPreview - Показывает, что в шалкере.

SwingAnimations - Меняет анимацию удара/использования.

Trails - Трейл за игроком.

ViewModel - Позволяет менять расположение левой и правой рук на экране.

World - Позволяет настроить время в мире на стороне клиента.

### Misc
ClientSounds - Прикольные звуки при нажатии кнопок.

DeathCoords - Птправляет координаты смерти в чат.

DiscordRPC - Показывает плашку в дискорде, что вы используете [вставьте название]визуалс.

FTHelper - Отображает задержки на кастомные предметы фантайма, позволяет забиндить кнопки на хотбар для использования этих предметов.

HWHelper - Отображает задержки на кастомные предметы холиворлда, позволяет забиндить кнопки на хотбар для использования этих предметов.

RWHelper - Ультра WIP.

IRC - Позволяет подключаться к IRC чату (хз зачем оно вообще здесь, айпи и ключ можно задать в IrcManager.class).

NameChanger - Меняет никнейм на стороне клиента.

NoCommands - Отключает настройку через команды.

Optimizer - Занимается чисткой утечек ОЗУ. Должен повышать производительность на слабых ПК.

ServerRPSpoof - Отключает загрузку серверного ресурспака.

Unlink - Отключает мод полностью.

## Полезные команды
c$config - Позволяет сохранить конфиг.

с$macro - Позволяет забиндить команду на кнопку.

с$bind - Позволяет забиндить функцию на кнопку.

c$irc - Позволяет настроить IRC.

## Сборка из сурсов
1. Откройте папку с исходными файлами в IntelliJ IDEA
2. Нажмите на "Build", для того, чтобы докачались все необходимые файлы
3. Зайдите в Gradle -> Tasks -> build -> build
4. Ваш собранный дистрибутив CheeseVisuals будет находиться в [директория проекта]/build/libs/[название мода].jar
